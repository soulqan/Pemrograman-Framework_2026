1. Routing Berbasis File: Karena struktur folder = alamat URL. Kamu cukup buat file di dalam folder pages/, dan Next.js otomatis menjadikannya rute tanpa perlu setting tambahan.

2. Next.js vs React (CRA): React standar merender halaman di browser pembeli (Client-side), sedangkan Next.js bisa menyiapkan halaman di server (Server-side/Static). Hasilnya, Next.js lebih cepat dan lebih bagus untuk SEO.

3. npm run dev: Menjalankan server lokal untuk tahap koding. Fitur utamanya adalah Fast Refresh, di mana perubahan kode langsung muncul di browser secara instan.

4. - dev: Digunakan saat masih ngoding.
- build: Digunakan saat aplikasi sudah jadi untuk mengompres dan mengoptimalkan kode agar siap di-upload ke internet.
